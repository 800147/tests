Notifications

Подписка на оповещения

Особенности, которые надо учитывать:

  • В некоторых браузерах оповещения запрещены в приватном режиме
  • В iOS и iPadOS оповещения доступны только если добавить сайт на домашний экран
  • Подписка на Push уже включает в себя запрос на разрешение уведомлений. Отдельно их можно не запрашивать.

Отправка оповещений из сервис-воркера

Кнопки ниже нужны для проверки работы Notification API и Service Worker. Оповещения, которые будут отображаться после нажатия на эти кнопки отправляются без участия Push API. Для проверки Push используйте Controller (ссылка на него появляется в момент подписки на оповещения)
